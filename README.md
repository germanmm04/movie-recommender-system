# Movie Recommender System 

##  Project Description

Movie Recommender System is a machine learning project that suggests movies based on user preferences using unsupervised learning techniques.

The system groups similar movies using clustering algorithms and provides personalized recommendations based on user taste patterns.

---

##  Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

---

##  Machine Learning Approach

- Unsupervised Learning
- Clustering (K-Means)
- Feature engineering based on movie attributes
- Similarity-based recommendation system

---

##  Main Features

- Movie grouping using clustering techniques
- Recommendation based on user preferences
- Data preprocessing and cleaning
- Visualization of clusters
- Simple and interpretable recommendation logic

---

##  Dataset

- Movie dataset containing metadata (genres, ratings, etc.)
- Preprocessed and cleaned for machine learning
- Feature selection for clustering
